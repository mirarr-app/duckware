2. Zero Data Collection

We value privacy as a core principle of software craftsmanship. DuckWare does not collect, record, track, or share any personal information. There are no registration forms, telemetry loggers, tracking cookies, or analytics scripts (such as Google Analytics) embedded in this site.

3. Local Browser Storage

This website does not run any backend servers to collect user settings. Any preferences or cached submissions remain strictly inside your browser's local sandbox using localStorage. No local storage data is ever transmitted to us or any third-party.

4. Static Hosting

DuckWare is hosted statically using GitHub Pages. GitHub may collect server access metadata (such as IP addresses and request logs) for security and operational purposes, in accordance with the GitHub Privacy Statement. We do not have access to these logs.
