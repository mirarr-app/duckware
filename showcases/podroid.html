What is Podroid?

Most "Linux on Android" solutions are user-space hacks — chroots, proot layers, or Termux wrappers that approximate a server environment. Podroid takes a different path: it boots a genuine Alpine Linux virtual machine with its own kernel, giving you container tooling that behaves the way it would on bare metal.

The result is a rootless Android app that turns any arm64 phone running Android 8 or newer into a portable dev box. Podman, Docker, and LXC ship pre-installed. An integrated terminal, X11 desktop viewer, SSH access, and port forwarding close the gap between a pocket device and a full Linux workstation.

Architectural Highlights & Strengths

  • Real Virtualization, Not Emulation Theater

    Podroid runs Alpine on a custom kernel via QEMU, with hardware-accelerated AVF on supported pKVM devices. Because the guest is a proper VM rather than a filesystem overlay, container runtimes, networking stacks, and privilege models work exactly as they would on a dedicated Linux host.

  • Containers Ready at First Boot

    Podman, Docker, and LXC arrive pre-configured inside the guest image. Spin up services, test deployments, or run ad-hoc workloads from the in-app terminal the moment the VM reports ready — no manual package hunting or daemon setup required.

  • Full-Stack Mobile Linux Experience

    Beyond CLI work, Podroid ships an xterm-256color terminal with over a hundred color themes, USB passthrough, SSH server support, guest-to-Android bridging, and a built-in X11 viewer for running GUI desktop applications with touch, keyboard, mouse, and audio input.

Getting Started

Install the APK, tap Start VM, and wait for the ready signal. From there the guest shell opens immediately to a familiar Alpine environment:

alpine shell
# Run containers without any extra setup
podman run --rm alpine echo "hello from a container"
docker run -d -p 8080:80 nginx

# Forward a guest port to your phone and LAN
podroid-forward add 8080 8080 tcp

# SSH in from a laptop on the same network
ssh root@<phone-ip> -p 9922

Building from Source

The entire stack — custom kernel, root filesystem, QEMU backend, and Android APK — can be rebuilt from a single script. Docker and the Android SDK/NDK are the main toolchain prerequisites:

bash
git clone https://github.com/ExTV/Podroid.git
cd Podroid
./build-all.sh all
