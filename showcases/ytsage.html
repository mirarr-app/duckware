CLI power, desktop manners.

YTSage sits on top of yt-dlp and FFmpeg, but the product is the window — paste, analyze, choose a row in the format table, and download without memorizing flags. That alone makes it useful for anyone who wants media archives without living in a shell.

Where it earns a showcase is the depth behind the calm UI: playlist selectors and exports, cookie login, proxy support, section trimming, subtitle merging, EBU R128 audio normalization, and a built-in updater for yt-dlp, FFmpeg, and Deno across fourteen languages.

Strength 01

Format table first

See resolution, FPS, HDR, codecs, and sizes before you commit — then lock video, audio-only, or converted formats.

Strength 02

Sponsor-aware captures

Strip SponsorBlock segments, burn or embed subtitles, save thumbnails and descriptions, and embed chapters in one pass.

Strength 03

Beyond a single video

Playlist queues with selective picking, Generic Mode for non-YouTube extractors, and packaged builds for Windows, Linux, and macOS.

Get it running

pip, or grab a packaged build

Install from PyPI when you want the Python entry point, or use the prebuilt Windows installers, Linux packages, and macOS apps from the latest release — FFmpeg-bundled options included.

$ pip install ytsage
