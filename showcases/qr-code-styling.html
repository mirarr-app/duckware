What is QR Code Styling?

Default QR generators produce identical black-and-white squares that look identical regardless of brand. For marketing pages, event tickets, or app onboarding flows, you want codes that carry your visual identity — rounded dots, gradient fills, corner styling, and a centered logo — without sending data to a third-party API.

qr-code-styling is a client-side JavaScript library that renders fully customized QR codes in the browser. Configure dot shapes, colors, background images, and embedded logos through a simple options object, then export to canvas or SVG for download or inline display.

Architectural Highlights & Strengths

  • Deep Visual Customization

    Control every visual layer independently: dot style (square, rounded, classy, extra-rounded), corner square and dot shapes, gradient fills, background color or image, and margin sizing. The result reads as a designed asset, not a generic barcode pasted onto your page.

  • Logo Embedding with Error Correction

    Drop a brand logo into the center of the code and the library handles the error-correction math so the QR remains scannable. Adjust logo size, margin, and cross-origin settings without manually tuning redundancy levels or worrying about unreadable output.

  • Framework-Agnostic, Client-Side Only

    Works in any JavaScript environment — vanilla JS, React, Vue, Angular, or Node.js for server-side rendering. No external API calls, no rate limits, no dependency on a hosted QR service. Generate codes entirely in your app and export as PNG, SVG, or raw canvas data.

Basic Usage

Instantiate with your data and style options, append to the DOM, and download or update on the fly:

index.js
import QRCodeStyling from "qr-code-styling"

const qr = new QRCodeStyling({
  width: 300, height: 300,
  data: "https://example.com",
  image: "/logo.png",
  dotsOptions: { color: "#4267b2", type: "rounded" }
})

qr.append(document.getElementById("canvas"))
