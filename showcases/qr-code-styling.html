Audit · what it is

A styling layer for scannable marks

Most QR libraries stop at black squares on white. QR Code Styling treats the matrix as a design surface: you pick data, drop in a logo, reshape the modules and corner finders, then paint with solid color or linear/radial gradients — all from a single options object in the browser.

There is no encoding API to call and no pixel pipeline to host. Instantiate QRCodeStyling, append to a DOM node, update when props change, and download PNG, JPEG, WebP, or SVG. Framework examples exist for React, Vue, Angular, Next, and Node when you need server-side buffers.

API surface

One constructor, full brand control

Configure once, append to the page, then update() or download() as the product UI changes. Extensions can post-process the SVG when you need borders or custom ornaments.

  • dotsOptions.type — module geometry
  • cornersSquareOptions / cornersDotOptions
  • image + imageOptions for center logos
  • gradient.colorStops on any painted layer
brand-qr.js options 
// Client-side branded QR — no network roundtrip
const qr = new QRCodeStyling({
  width: 300,
  height: 300,
  type: "svg",
  data: "https://example.com/join",
  image: "/logo.svg",
  dotsOptions: {
    type: "rounded",
    gradient: {
      type: "linear",
      colorStops: [
        { offset: 0, color: "#00f5ff" },
        { offset: 1, color: "#ff00aa" }
      ]
    }
  },
  cornersSquareOptions: { type: "extra-rounded" },
  backgroundOptions: { color: "#0a0a0f" }
});

qr.append(document.getElementById("canvas"));
qr.download({ name: "brand-qr", extension: "svg" });

Why it matters

QR as brand furniture

Packaging, tickets, and onboarding flows increasingly put a QR front-and-center. When that mark looks generic, the whole surface feels unfinished. Shipping styled codes from the same frontend that owns the brand tokens keeps print and product UI in sync — and keeps sensitive URLs off a third-party generator.

Strengths

Three reasons it sticks

01 · privacy & speed

No API tax

Encoding and drawing happen locally. Campaign sites and SPAs can mint codes offline-capable and without leaking payload strings to a QR SaaS.

02 · design depth

Layered style model

Dots, corner squares, corner dots, background, and logo each take color or gradient — enough control to match a design system without redrawing the matrix by hand.

03 · delivery formats

Canvas or SVG out

Raster for quick shares; vector for print and crisp UI. getRawData and download cover the export path in both browser and Node setups.
