// about

Web downloaders trade convenience for trust — every page is a maze of deceptive buttons and redirect chains. Running yt-dlp raw works, but format strings and merge flags are a chore when you just want one clean file.

yoinks occupies the middle ground: paste a URL from YouTube, X, Instagram, Threads, or TikTok, browse resolutions with estimated sizes, pick audio-only MP3 if you want, and land the file in ~/Downloads. It takes over the terminal viewport, centers the UI, and hands your scrollback back when you quit.

// interface

// why it stands out

01

Arrives with its own engine

First run pulls a standalone yt-dlp binary into ~/.yoinks/bin — no Python layer. Existing installs are reused. ffmpeg on PATH handles merges; ffmpeg-static covers the fallback.

02

Terminal-native interaction

Arrow keys, j/k, number shortcuts, or mouse clicks on the yoink button and format list. ^t cycles auto, light, and dark themes that inherit your shell colors by default.

03

Choose before you download

The format picker shows codecs, resolution labels, and approximate file weights side by side — including MP3 extraction. Split high-res streams merge through ffmpeg automatically after you pick once.

// install
