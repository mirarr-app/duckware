What is yoinks?

Most web-based video downloaders are a minefield of deceptive buttons, redirect chains, and ad overlays. Power users reach for yt-dlp directly, but memorizing format codes and merge flags is not everyone's idea of a good time.

yoinks wraps the extraction engine in a full-screen Ink terminal UI: paste a URL from YouTube, X, Instagram, TikTok, or any of 1,800+ supported hosts, browse available resolutions with estimated file sizes, and save to ~/Downloads. The experience is interactive when you want it and scriptable flags are on the roadmap — but the core promise is honest downloads without the sketchy web middleman.

Architectural Highlights & Strengths

  • Self-Provisioning Download Stack

    On first launch, yoinks fetches a standalone yt-dlp binary into ~/.yoinks/bin — no Python install required. If a system copy already exists, it defers to that. For stream merging and MP3 extraction, it prefers ffmpeg on your PATH and falls back to the bundled ffmpeg-static package. The tool arrives ready to run with a single npm install -g or npx invocation.

  • Full-Screen Ink UI with Dual Input

    Built on React-for-the-terminal via Ink, yoinks takes over the viewport, centers its panels, and restores scrollback on exit. Navigate formats with arrow keys, vim bindings, number shortcuts, or mouse clicks on the yoink button, format list, and footer controls. The auto theme inherits your terminal's native colors; ^t cycles through light and dark palettes for the session.

  • Informed Format Selection

    Before committing bandwidth, the format picker surfaces resolution labels, codecs, and approximate file sizes side by side — including an audio-only MP3 path. High-resolution streams that require separate video and audio tracks are merged automatically via ffmpeg, so you pick once and yoinks handles the post-processing pipeline.

Getting Started

Requires Node 18+. Install globally or try instantly without a permanent install:

terminal
npm install -g yoinks
# or: npx yoinks
