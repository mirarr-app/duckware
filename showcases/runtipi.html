Runtipi

A personal homeserver for everyone.

One install command, a clean web dashboard, and one-click apps — Docker orchestration without the YAML ritual.

Get started Try the demo Source

Getting on the box

One command. Then tipi.local.

Runtipi targets everyday 64-bit Linux hosts. Pipe the installer, land on a dashboard, and stop babysitting compose files for every media stack and utility you want to run.

bash — setup.runtipi.io
$ curl -L https://setup.runtipi.io | bash

Why it sticks

Homelab power without the ceremony.

Built in TypeScript with NestJS and React, Runtipi sits between raw Docker and heavyweight PaaS — opinionated enough for weekends, open enough for people who still want to edit compose when it matters.

01 — install

Click, configure, run

Browse the official store (and community stores), fill the few fields an app needs, and let containers come up with networking handled for you.

02 — control

Dashboard as control room

Watch status, update with one click, back up before you pull the trigger, and keep overrides that survive upgrades.

03 — open

GPL-3.0 and community-shaped

Volunteer-maintained, contribution-friendly, and designed so new apps plug in without rewriting the orchestrator.
