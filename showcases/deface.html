Deface

Automatic face anonymization for videos and photos — detect, blur, and export from the command line.

What is Deface?

Publishing field recordings, dashcam footage, or research video often means redacting bystanders by hand — frame by frame in an editor, or hoping a blur rectangle stays aligned as people move. For batch workflows or privacy-sensitive datasets, that manual step does not scale.

deface is a Python CLI that detects human faces in each video frame or still image, then applies an anonymization filter to every detected region. Point it at a file path, a directory of media, or a live webcam feed and get an output with blurred ellipses, black boxes, mosaics, or custom replacement images — no GUI required.

Architectural Highlights & Strengths

  • CenterFace Detection, Frame by Frame

    Built on the CenterFace neural network trained on the WIDER FACE dataset. Each frame is analyzed independently for robust detection across scales, poses, and partial occlusions. Bounding boxes drive ellipse masks or rectangular overlays wherever a face is found.

  • Flexible Anonymization Modes

    Default output applies strong Gaussian blur inside elliptical face regions. Switch to solid black boxes, mosaic tiles, a custom image, or no filter at all. Tune detection thresholds, mask scale, and downsample resolution for high-res inputs without sacrificing throughput.

  • Batch Processing and Hardware Backends

    Process single files, glob patterns, or entire directories in one command. Optional ONNX Runtime backends offload inference to CUDA, DirectML, or OpenVINO when available. Live preview mode works with attached webcams for quick testing before batch runs.

Basic Usage

Install via pip, then pass a video or image path. Output defaults to the input filename with an _anonymized suffix:

terminal
python3 -m pip install deface

deface myvideos/vid1.mp4
# → myvideos/vid1_anonymized.mp4

deface cam
# live webcam demo
