影 · shadow kage

Freeze a living site into a quiet, script-free copy.

A Go tool that opens pages in real Chrome, waits for them to settle, then tears every script out — leaving HTML you can keep offline for years.

View on GitHub Documentation Releases

“Save As” was never yours. kage takes the finished page — then keeps only the shadow.

映

Render in Chrome

Headless Chrome loads each page for real. Spinners finish, lazy images arrive, the DOM settles into what a person would have seen.

削

Strip every script

Handlers, analytics, and javascript: links are removed. What remains cannot phone home — CSS, images, and fonts are pulled onto local paths.

蔵

Keep the mirror

A polite crawl respects robots and sitemaps, resumes after Ctrl-C, and writes a folder you can serve, share, or pack into one file.

Quick start

›kage clone paulgraham.com
›kage serve ~/data/kage/paulgraham.com
# open http://127.0.0.1:8800

Needs Chrome or Chromium on the host — or run the container image with Chromium already inside.

Carry the whole site as one artifact

A folder is fine for reading. For travel, pack the mirror into an open archive, a self-serving binary, or a double-click app.

  • .zim
    Open ZIM archive Deterministic, Kiwix-compatible. Same mirror always yields the same bytes — open in kage or across the Kiwix apps.
  • binary
    Self-serving executable The archive glued to kage. Send one file; the recipient needs nothing installed. Cross-pack with a release --base.
  • app
    Desktop shell macOS .app, Linux AppImage-style bundles, or a Windows GUI viewer — double-click, no terminal flash.
