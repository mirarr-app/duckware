PIXEL SNAPPER

Messy AI pixels in. Perfect grid out. A Rust tool that detects cell size, snaps every texel onto a crisp lattice, and locks colors to a quantized — or fully custom — palette.

THE DRIFT PROBLEM

AI paints pixels. It does not honor grids.

Generative models can fake a retro look, but the underlying raster is soft: cell sizes wander, edges anti-alias, and colors smear outside any usable game palette. That output breaks tilemaps, scales into mush, and forces artists to redraw by hand.

Sprite Fusion Pixel Snapper was built for that gap. Feed it a PNG or JPEG (or a whole folder), and it auto-detects pixel size, aligns every cell to a uniform lattice, and quantizes color — defaulting to 16 colors or accepting a comma-separated hex palette you define. Dithering and fine texture survive the pass whenever the signal is there.

SNAP STRENGTHS

Three reasons the lattice holds

Cell detection + override

Auto-finds the dominant pixel size, or pin it with --pixel-size when batch sheets share a known scale. Output stays resolution-consistent and scales cleanly.

Palette quantization

Reduce to k colors or lock to an exact custom palette. Same colors across a sprite sheet — no surprise hues sneaking in from the model.

Detail-aware pass

Designed to keep dither patterns and micro-contrast instead of flattening everything into muddy blobs. Usable assets, not just “prettier” screenshots.

Example --palette lock from the README — eight hex stops, zero drift.

THREE SURFACES

CLI · WASM · Web — one snap core

Ship the algorithm wherever your art pipeline lives. Native binary for batch jobs, WebAssembly for embedding, and a hosted app when you just need to drop a file and download the fix.

CLI

Brew / Cargo binary

Install via Homebrew or cargo install, then point at a file or directory. Built for sprite-sheet cleanup loops and CI-friendly batch runs.

WASM

Embed process_image()

wasm-pack the crate, call process_image(bytes, k?, size?, palette?) from JS, and keep snapping inside your own editor or exporter.

WEB

spritefusion.com app

No install — open the online Pixel Snapper (or the desktop edition) when you want a visual pass before committing assets.

spritefusion-pixel-snapper — example invocations
# single image → 16-color snap
spritefusion-pixel-snapper input.png output.png 16

# batch a folder with fixed cell size
spritefusion-pixel-snapper sprites/in sprites/out 16 --pixel-size 8

# lock an exact palette
spritefusion-pixel-snapper input.png output.png --palette "0d2b45,203c56,544e68,8d697a"

ZOOM ON DETAIL

What survives the snap

Beyond coarse alignment, Pixel Snapper aims to retain the texture that makes pixel art feel hand-authored — including dithering — so cleaned sprites still read as intentional, not posterized noise.
