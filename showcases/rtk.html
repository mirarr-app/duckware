rtk

A transparent CLI proxy that shrinks command output before it floods your LLM context.

Visit Website View Source

The Core Idea

agent session
# Without rtk — raw git diff floods context
git diff → 10,000 tokens

# With rtk — filtered, structured summary
rtk git diff → 2,500 tokens (-75%)

What is rtk?

Coding agents like Claude Code spend a large fraction of their context window on command output — verbose test logs, thousand-line diffs, directory listings, linter reports. You pay for every token whether the model needs it or not. rtk (Rust Token Killer) sits between your shell and the agent as a transparent proxy, intercepting stdout from common dev commands and returning a compressed, structured version that preserves what matters.

It is a single Rust binary with no runtime dependencies, supports over 100 commands out of the box, and adds less than 10ms of overhead per invocation. In a typical 30-minute agent session, users report roughly 80% fewer tokens consumed across git, test, lint, and filesystem operations.

Architectural Highlights & Strengths

  • Command-Aware Compression

    rtk does not apply generic gzip-style shrinking. Each supported command — git diff, cargo test, rg, pytest, docker ps, and dozens more — has a tailored filter that extracts signal and discards noise. A test run returns pass/fail summaries instead of full stack traces; a diff returns changed hunks instead of entire files.

  • Drop-In Proxy, Zero Workflow Change

    Prefix commands with rtk or configure your agent to route through it automatically. The underlying tool still runs normally — rtk only transforms what comes back. No plugins, no API keys, no changes to your project. Install via Homebrew or a one-line curl script and you are running in seconds.

  • Single Binary, Native Speed

    Written in Rust and shipped as one self-contained executable, rtk avoids the dependency chains that make other dev tools fragile to install. Sub-10ms per-call overhead means compression pays for itself immediately — the tokens you save on a single cargo test output dwarf the proxy cost.

Installation

Install via Homebrew on macOS or the quick-install script on Linux:

bash
brew install rtk

# or on Linux/macOS
curl -fsSL https://raw.githubusercontent.com/rtk-ai/rtk/refs/heads/master/install.sh | sh
