ZenNotes

Keyboard-first local Markdown notes with Vim motions, databases, and MCP integration.

Visit Website View Source

What is ZenNotes?

ZenNotes is a local-first, keyboard-centric note workspace designed to help developers write and organize plain Markdown vaults. Rather than trapping content inside proprietary databases, ZenNotes acts as a performant browser and editor overlay directly on ordinary .md text files on disk.

It features Vim keybindings, CodeMirror 6 live preview editing, outline jumps, LaTeX math, and Mermaid diagrams. It also introduces Notion-style tables and boards overlays directly over plain .csv files, blending databases with flat-file structure.

Architectural Highlights & Strengths

  • Multi-Runtime Portability

    ZenNotes ships from a monorepo containing a shared React core. It deploys natively as an Electron-packaged desktop client (signed and notarized for macOS/Windows/Linux), as a self-hosted Go-backend web client, or as a Docker container, making it ideal for home servers and private LAN networks.

  • Obsidian-Friendly & Asset Agnostic

    Recognizes standard Markdown linkages, loose image assets, daily/weekly notes directories, and embeds. It interfaces naturally with existing Obsidian-style flat vaults, resolving images and wiki-links directly at the root folder.

  • Built-in MCP Integration

    A standout feature for AI pair-programming: the app includes a first-party Model Context Protocol (MCP) server. This exposes vault utilities (list, search, read, write, and task checks) directly to compatible AI models (Claude Code, Claude Desktop, Codex) to let assistants interact directly with your notes.

Command Line Companion

The desktop app registers a companion zen CLI, facilitating terminal searches, edit commands, and Raycast launcher search extensions natively on macOS:

terminal
# Query the local note vault from your CLI
zen search "mcp server"
Found 2 matches in daily-notes/2026-06-21.md

Installation

Installers for macOS and Windows are attached to GitHub releases. Linux users can install natively or via the AUR:

bash
# Install on Arch / Manjaro via AUR
yay -S zennotes-bin
