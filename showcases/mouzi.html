What is Mouzi?

The Downloads folder is where every installer, PDF, screenshot, and zip file lands — and where most of them stay forever. Manual cleanup works until it does not, and cloud-based organizers trade convenience for sending filenames (or worse) to someone else's server.

Mouzi is a free, privacy-first desktop app built with Tauri and Rust that lives in the system tray and quietly sorts watched folders as files arrive. Images, documents, archives, installers, and media each route to dedicated subfolders via customizable rules — with per-folder silent, manual, or paused modes, full undo history, and zero cloud dependency.

Architectural Highlights & Strengths

  • Silent Background Organization

    Runs 24/7 at roughly 5 MB RAM, watching folders with the Rust notify crate and applying the first matching rule within seconds of a new file landing. A subtle toast reports how many files were organized — no modal dialogs interrupting your workflow.

  • Rules Engine with Ignore Patterns

    Default rules cover common file types out of the box. Extend with regex patterns, dynamic path placeholders like {year}/{month}, and per-folder .mouziignore files. Google Takeout archives can be imported and sorted in one step without manual unzip-and-drag.

  • 100% Offline, Locally Stored

    No telemetry by default, no filename uploads, no cloud sync. Action history lives in a local SQLite database with one-click undo. Available for Windows and Linux as installer, MSI, portable, AppImage, deb, or rpm — with a portable build that leaves no registry trace.

Quick Start

Install, let Mouzi start with the tray icon, and drop a file into Downloads — it routes to the right subfolder within a couple of seconds:

workflow
1. Install Mouzi → tray icon appears
2. Drop invoice.pdf into Downloads/
→ Downloads/Documents/invoice.pdf

Left-click tray # recent actions & stats
Right-click tray # Clean Now, Settings, Quit
