The gap it fills

Clipboard sync without the vendor tax

Most “universal clipboard” options either lock you into one OS ecosystem or route every paste through a cloud service you have to trust with passwords, tokens, and screenshots. Mixed Windows/Linux desks, searchable history, and auditable crypto rarely show up in the same product.

UniClipboard treats the clipboard as a shared encrypted space: devices join with an invitation code and passphrase, talk peer-to-peer when they can, and fall back to a relay that never holds plaintext. Text, images, and files move with you — history stays encrypted at rest, including the search index.

Path of a paste

Copy once. Arrive encrypted.

Payload crypto is independent of transport. Whether the bytes travel direct or through a fallback relay, only your devices hold the keys.

01

Copy

Ctrl/Cmd+C on any paired desktop — text, image, or file.

02

Encrypt

XChaCha20-Poly1305 AEAD; keys from Argon2id + system keyring.

03

Punch / relay

Direct P2P or NAT hole-punch; encrypted relay if the path fails.

04

Decrypt

Only space members can open the payload — never the relay.

05

Paste

Arrive on the other machine (or phone companion) ready to paste.

Why it sticks

Three signals that matter

P2P first, relay last

Pairing is a one-time public-key exchange — no email signup. Same Wi-Fi or cross-network hole-punching keeps traffic on your path when possible. Sleep, Wi-Fi flips, and brief drops recover without re-pairing.

Crypto you can name

XChaCha20-Poly1305 for payloads, Argon2id for passphrase derivation, master key in the OS keyring, per-space isolation so switching spaces re-encrypts local history. A hostile relay still only sees ciphertext.

Quick Panel, CLI, phone

A keyboard overlay previews clips without leaving your flow. The uniclip CLI mirrors the GUI for SSH and scripts. iOS and Android join as companions with QR pairing — share a phone screenshot to the desk in seconds.

In the wild

Desktop sync, visible

History, devices, and the quick panel sit in a Tauri desktop shell — built to feel like part of the OS clipboard, not another tab you babysit.

Get a space running

Install, invite, join

Create a space on the first device, then invite others with a short-lived code — or drive the same flow from the terminal.

uniclip · headless flow AGPL-3.0
$ curl -fsSL https://uniclipboard.app/install.sh | bash
$ uniclip init # create encrypted space
$ uniclip invite # short-lived join code
$ uniclip join <code>
