# DuckWare

A curated catalog celebrating exceptional software craft. This repository hosts a quiet, minimalist directory of software and developer tools that stand out for their utility, performance, or design.

## Project Details

- **Purpose**: A personal collection of cool software and developer tools found across the web.

- **Website URL**: [https://mirarr-app.github.io/duckware](https://mirarr-app.github.io/duckware)
- **Showcases Catalog**: [https://mirarr-app.github.io/duckware/showcases](https://mirarr-app.github.io/duckware/showcases)

## Curation Rules

If you wish to submit an application for audit, please open an issue in this repository describing:
1. The project's repository or website URL.
2. What makes it special (performance highlights, local-first architecture, custom styling support, etc.).
